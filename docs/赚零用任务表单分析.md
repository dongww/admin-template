#转领用任务表单分析
##基本信息

|表单项        |类型      |说明|
|--------------|----------|----|
|任务图片      |text      |    |
|项目编号      |text      |    |
|任务名称      |text      |     |
|任务描述       |textarea    |   |
|是否含有视频  |bool      |       |
是否上传音频		bool
是否嵌套爱调研问卷	bool
是否冻结（及冻结天数）	bool number
奖励类型		二选一，不同选项对应的额外表单项有所不同
审核通过一份奖池增加	number 元
提交后是否立即抽奖	bool	若选“是”，则需填写额外的表单
问卷分组格式		多选一
获奖用户确定方式	多选一	不同选项对应不同额外选项
其他备注		textarea
是否过滤		bool
任务类型		三选一 不同选项对应不同额外选项
