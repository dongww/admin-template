介绍

及至极客管理后台界面模板框架 是一套基于 AdminLTE， 结合团队自身和项目实际情况而定制开发的后台界面模板框架。

开发目的：

  • 使团队的产品后台具有一致的界面
  • 可复用的资源和工具，可以减少工作量，提高效率
  • 团队统一使用一套工具，便于相互协作。若长期使用，可以不断积累经验并完善工具，形成良性循环
  • 使后端人员可以基本脱离前端工程师而独立开发适用于大多数场景的美观后台界面

开发目标：

  • 一致的风格
  • 独立于后端，任何后端语言皆可配合使用
  • 以组件化形式搭建后台管理界面，减小使用难度，后端人员容易上手
  • 尽可能多地提供各类场景下的界面示例，避免开发人员重复排版，并保持统一风格
  • 提供主要的前后端通讯机制（AJAX等），并易于使用

目录结构

    |--index.html  索引页
    |--docs        说明文档
    |--css         样式表目录
    |--image       图片目录
    |--js          Javascript 目录
    |--pages       包含所有演示页面
        |--content 各种以文本或图文展示性内容为主的演示页面
        |--form    各种表单相关的演示页面
        |--list    各种列表形式为主的演示页面
        |--home    各种管理后台首页的演示页面
        |--example 各种特殊场景、综合类及较复杂的演示页面
